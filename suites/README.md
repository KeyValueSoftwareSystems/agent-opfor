# Suites (source of truth)

Suite definitions reference evaluator IDs under `evaluators/{agent|mcp}/`.

| Path            | Use                      |
| --------------- | ------------------------ |
| `suites/agent/` | Agent red-teaming suites |
| `suites/mcp/`   | MCP red-teaming suites   |

After edits, run `npm run sync:skills-evaluators`.
